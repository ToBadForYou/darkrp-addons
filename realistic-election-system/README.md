# Realistic election system for DarkRP
An election system modification for Garry's Mod written in 2018, the code is very old and outdated but should still be functional. Feel free to use and modify the code as you wish but remember to give credit. 

It used to be sold on gmodstore and you can view more details [here](https://www.gmodstore.com/market/view/5794)

Requires version 2.5.0+ version of DarkRP, tbfy_shared_v2 and CSS content to be installed
